# Weather Time Machine

Visualisez et comparez l'historique météo de n'importe quelle ville sur plusieurs années.

## Stack

- React 19 + Vite
- Tailwind CSS v4
- Recharts
- Open-Meteo API (gratuite, sans clé)

## Fonctionnalités

- Recherche de ville avec autocomplétion
- Sélection de période (1940 à aujourd'hui)
- Dashboard KPI (records, tendances, moyennes)
- 6 types de graphiques interactifs
- Mode comparaison de 2 villes
- Dark/Light mode
- Français / Anglais

## Développement

```bash
npm install
npm run dev
```

## Build & Déploiement

```bash
npm run build
```

Déploiement automatique sur GitHub Pages via GitHub Actions.

## Crédits
